# VulpOS

A Linux-based operating system built from scratch with natural language commands and GitHub-based package management.

## Current Status: Phase 2 Complete
- ✅ Linux kernel boots
- ✅ Custom init system
- ✅ Echo shell with natural language shutdown command

## Building
```bash
./build.sh
```

Or use VS Code: `Ctrl+Shift+B`

## Project Structure
- `init.c` - Init system (PID 1)
- `shell.c` - Shell with natural language commands
- `build.sh` - Build and test script
- `custom-os-project-plan.md` - Full project roadmap
